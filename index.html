🔧 Google API Setup

⚠️ Do NOT open index.html directly. Google OAuth blocks file:// URLs.
You must serve the app over HTTP first (see Step 1 below).
  1. Serve the app via HTTP. Open Terminal and run:
    cd /Users/ullaskumar/commissioning_app
    python3 -m http.server 8080
    Then open http://localhost:8080 in your browser.
  2. Go to console.cloud.google.com → create or select a project
  3. Enable Google Sheets API and Google Drive API
  4. Go to APIs & Services → Credentials
  5. Create OAuth 2.0 Client ID (type: Web application)
  6. Under Authorized JavaScript origins add:
    http://localhost:8080 (for local use)
    or your hosted URL if deploying online
  7. Under OAuth consent screen → add your Gmail as a Test user
  8. Copy the Client ID and paste it into the field on the main page
The app uses drive.file and spreadsheets.readonly OAuth scopes. Station data is read from the master Google Sheets file, and the commissioning report is saved as a Google Doc in the same Drive folder.
